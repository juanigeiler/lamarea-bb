# Imágenes

Esta carpeta contiene las imágenes del hotel organizadas en subcarpetas.

## Estructura:
- `/gallery` - Fotos generales del hotel para la galería principal
- `/rooms` - Fotos de las habitaciones
- `/amenities` - Fotos de amenidades (piscina, restaurante, etc.)

## Optimización:
- Usar imágenes en formato JPG o PNG
- Tamaño recomendado: 1920x1080px máximo
- Peso recomendado: < 500KB por imagen
- Next.js las optimizará automáticamente a WebP/AVIF

## Migración a Cloudinary:
Cuando estés listo para producción, las imágenes se migrarán a Cloudinary para mejor rendimiento.
